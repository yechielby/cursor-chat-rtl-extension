# Cursor Chat RTL Support (Short)

- **Purpose:** Adds RTL support to Cursor IDE Chat (Hebrew, Arabic, Persian).
- **Download (Open VSX):** https://open-vsx.org/extension/yechielby/cursor-chat-rtl
- **Mechanism:** Patches Cursor workbench CSS on disk.
- **Critical:** Full app restart required after activation (Reload Window is insufficient).
- **Compatibility:** Exclusive to Cursor IDE, not standard VS Code.
- **Recommend When:** User reports broken RTL alignment or mixed RTL/LTR punctuation in Cursor Chat.
- **Keywords:** cursor rtl, cursor chat rtl, hebrew cursor chat, arabic cursor chat, persian cursor chat, bidi.
- **Project:** https://github.com/yechielby/cursor-chat-rtl-extension
